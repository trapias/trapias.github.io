Back to DNN Modules

DNN Module

Overview

XSLideShow is a free DotNetNuke module for publishing photo galleries and slideshows. It provides an easy way to showcase images on your DNN website with various display options and transitions.

Features

  • Create beautiful image galleries
  • Automatic slideshow with configurable transitions
  • Thumbnail navigation
  • Responsive design
  • Lightbox support
  • Multiple layout options
  • Easy image upload and management

Supported Formats

XSLideShow supports all common image formats including JPEG, PNG, GIF, and WebP.

Download

XSLideShow is open source and available on GitHub.
