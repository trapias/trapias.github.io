DNN Module

Overview

ODSForm is a DotNetNuke module that allows you to publish simple feedback forms with customizable fields. It's perfect for contact forms, surveys, and other data collection needs.

Features

  • Create custom forms with various field types
  • Email notifications on form submissions
  • Store submissions in database for later review
  • Customizable field validation
  • Anti-spam protection
  • Export submissions to Excel/CSV

Field Types

ODSForm supports various field types including:

  • Text fields
  • Text areas
  • Dropdown selects
  • Checkboxes
  • Radio buttons
  • Date pickers

Download

ODSForm is open source and available on GitHub.
