DNN Modules

A collection of modules I developed for DotNetNuke (DNN) CMS platform. These are open source and freely available.

DNNPiwik

A DotNetNuke module to easily integrate Piwik tracking script in any page, allowing to exclude certain roles from tracking, tipically Administrators.

Learn more

EasyWiki

An extensible wiki module for DotNetNuke built on top of XsltDb.

Learn more

ODSForm

A DotNetNuke module that lets you publish simple feedback forms with a variable list of custom fields.

Learn more
